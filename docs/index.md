# FanThemeFCNantes Documentation

> **Disclaimer:** This is an independent fan project. It is not affiliated with, endorsed by, or connected to FC Nantes in any way.

## Installation

1. Copy the `FanThemeFCNantes` folder to your Matomo `plugins/` directory
2. Go to **Administration > Platform > Themes**
3. Click **Activate** next to "FanThemeFCNantes"

## What it changes

### Colors

The theme applies FC Nantes' canari yellow and green palette across the entire Matomo interface:

| Element | Color |
|---------|-------|
| Header background | Dark green (#1A472A) |
| Header text | Canari yellow (#FCD209) |
| Page background | Warm yellow-white (#FDFBE8) |
| Text | Dark green (#0E2E1A) |
| Links | Pitch green (#1A6B35) |
| Borders | Soft gold (#D4CE8A) |
| Focus ring | Canari yellow (#FCD209) |

### Graph palette

8 data series alternate between green shades and gold tones to match the FC Nantes identity.

### Background

An illustrated aerial view of the Stade de la Beaujoire is rendered as a transparent watermark (15% opacity) behind the dashboard, giving a subtle football atmosphere without interfering with readability.

### Visitor map

The visitor map uses a green-to-gold gradient with canari yellow highlights for selected and hovered countries.
