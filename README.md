# Fan Theme FC Nantes

## Description

Show your support for FC Nantes with this unofficial fan-made canari-themed Matomo dashboard. Features the iconic yellow and green color palette, a transparent illustration of the Stade de la Beaujoire as a subtle background watermark, and custom graph colors matching the club identity.

**Disclaimer:** This is an independent fan project. It is not affiliated with, endorsed by, or connected to FC Nantes in any way.

### Features

- **Canari yellow and green color scheme** across the entire interface — header, sidebar, graphs, maps, and sparklines
- **Stade de la Beaujoire background** — an illustrated aerial view of the stadium rendered as a transparent watermark behind the dashboard
- **Custom graph palette** — 8 data series alternating between FCN green shades and gold tones
- **Themed visitor map** — green-to-gold gradient with yellow highlights for selected countries
- **Clean card layout** — white content cards over the subtle stadium background

## FAQ

**How do I activate the theme?**

Go to Administration > Platform > Themes and click Activate next to "FanThemeFCNantes".

**Does this theme modify any data or tracking?**

No. This is a purely cosmetic theme. It only changes colors and adds a background image.

**Which Matomo versions are supported?**

Matomo 5.x is supported.
